# Installation sur iPhone

1. Ouvrir Safari et naviguer vers l'URL de l'application
2. Appuyer sur le bouton **Partager** (icône carré avec flèche vers le haut)
3. Défiler vers le bas et appuyer sur **Sur l'écran d'accueil**
4. Confirmer en appuyant sur **Ajouter**

L'application apparaît sur l'écran d'accueil et s'ouvre en plein écran.

## Utilisation

1. Exporter un fichier `.FIT` depuis Garmin Connect (via l'app ou le site web)
2. Ouvrir l'application Garmin Dive
3. Importer le fichier via le bouton ou par glisser-déposer
4. Les plongées sont sauvegardées localement dans le navigateur
